# 🏦 Bank Loan Risk Analytics Dashboard

## 📌 Project Overview
This Power BI project analyzes bank loan applications, funded amounts, repayment performance, and loan risk segmentation.  
The dashboard helps stakeholders monitor loan quality, track financial KPIs, and identify high-risk loan categories.

---

## 🎯 Business Objectives
- Analyze overall loan application trends
- Monitor funded vs received loan amounts
- Identify good and bad loan ratios
- Evaluate repayment performance
- Understand loan distribution across states, purposes, and home ownership categories

---

## 📊 Key KPIs
| KPI | Value |
|------|------|
| Total Loan Applications | 38.6K |
| Total Funded Amount | $435.8M |
| Total Amount Received | $473.1M |
| Average Interest Rate | 12.0% |
| Average DTI | 13.3% |
| Good Loan Ratio | 86.2% |
| Bad Loan Ratio | 13.8% |

---

## 📈 Dashboard Pages

### 1️⃣ Summary Page
High-level overview of:
- Loan applications
- Funded amount
- Amount received
- Good vs bad loan segmentation
- Loan status analysis

### 2️⃣ Overview Page
Detailed trend and segmentation analysis:
- Monthly amount received trends
- State-wise loan distribution
- Loan purpose analysis
- Employee length analysis
- Home ownership segmentation

### 3️⃣ Details Page
Granular loan-level data including:
- Loan ID
- Purpose
- Grade & sub-grade
- Interest rate
- Installment amount
- Funded amount
- Amount received

---

## 🧠 Key Insights
- Majority of issued loans are classified as good loans (86%)
- Debt consolidation is the leading loan purpose
- Mortgage holders represent a significant share of borrowers
- Monthly repayment trends show consistent growth throughout the year
